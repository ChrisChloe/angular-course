Limpeza de Cache

Mas o que é a memória cache?
Simplificando: a memória cache é um conjunto de dados que o seu navegador vai guardando para tornar o acesso às páginas da web mais rápido. Toda vez que você acessa uma url, ao invés de carregar todos os elementos da página novamente, o navegador vai buscar na cache o que já tem salvo a fim de acelerar o carregamento da página.

E como limpar a memória cache?
A tecla F5 ou as combinações ctrl+F5 ou shift+F5 ou ctrl+R são atalhos padrão para limpeza da cache da maioria dos navegadores. No entanto, a utilização destes atalhos não elimina os dados do histórico de acesso às páginas.
